Maven commands

1. mvn clean - cleans /target
2. mvn compile - builds /target
3. mvn test - runs unit tests
4. mvn package - Packages compiled code into a JAR/WAR file inside target/
5. mvn install - installs required packages into our machine
6. mvn deploy - Deploys the package to a remote repository
7. mvn spring-boot:run - Run a Spring Boot application
8. mvn validate - Validates the project structure and pom.xml
9. mvn clean install - Fresh build
10. mvn install -DskipTests - skips test during build
